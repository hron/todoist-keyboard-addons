Todoist Keyboard Add-ons
=========================

A Chrome extension that adds the following keyboard shortcuts:

- Alt+Shift+Up — Move the focused task up (simulates drag-and-drop)
- Alt+Shift+Down — Move the focused task down
- Alt+Up — Go to the parent task when viewing a subtask
- Alt+O — Open the "More actions" menu in modal headers (layout-independent using event.code)
- Alt+K — Open the first link in the focused task's content, if one is present
